# Практична робота №6: Система Модальних Вікон
**Виконав:** Донець Костянтин, група Ік-41  
**Варіант:** 3 (Modal System)

### Реалізований функціонал:
1. Всі 5 типів модалок (Alert, Confirm, Form, Image Gallery, Video) з унікальними анімаціями (fade, slide, zoom).
2. Повний стек модалок (можливість відкривати вікно поверх іншого без багів).
3. Сувора доступність (Accessibility): реалізовано Focus Trap всередині відкритого вікна, навігація за допомогою Tab, ESC та стрілок.
4. Оптимізація через Event Delegation (слухачі подій надійно працюють на батьківських елементах).
